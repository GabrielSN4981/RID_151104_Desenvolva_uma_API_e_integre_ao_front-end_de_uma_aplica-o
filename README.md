# projeto-react-api-node
📚 Projeto CRUD de Livros

Aplicação full-stack com:

Frontend em React (Vite)

API em Node.js + Express

Banco de dados PostgreSQL

🚀 Executar o projeto

Instale as dependências:

npm install

Inicie o projeto (front + backend):

npm run start

⚠️ O backend deve rodar na porta 3433.

🌐 Deploy

Frontend: https://api-integrada-frontend.netlify.app
